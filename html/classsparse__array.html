Detailed Description

template<typename Component>
class sparse_array< Component >

A sparse array data structure for storing components in an ECS.

This template class provides an efficient way to store and access components associated with entities, allowing for optional values and dynamic resizing.

Template Parameters
ComponentThe type of the component to store in the sparse array.
